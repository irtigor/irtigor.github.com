Erro Ao Tentar Instalar O RedCloth Usando Gem

Aug 13th, 2011

Precisei instalar o modulo RedCloth, é uma dependência do octopress, o problema é que ele usa a cflag Werror pra tratar alertas como erro… cortando o lero-lero:

gem install RedCloth -- --with-cflags=\"-O2 -pipe -march=native -Wno-unused-but-set-variable\"

Fonte: http://stackoverflow.com/questions/6761670/is-there-any-way-to-change-gcc-compilation-options-for-a-gem
